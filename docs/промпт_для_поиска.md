{

  "name": "Полное название ресурса",
  "description": "Краткое описание (1-2 предложения)",


  "type": "тип_ресурса",
  "format": "очно / онлайн / выездное / смешанное_очно_онлайн",
  "duration": "длительность (часы/дни/месяцы)",


  "targetAudience": ["8 класс", "9 класс", "10 класс", "11 класс"],
  "subjects": ["Математика", "Экономика", ...],


  "benefits": {
    "bvi": true/false,
    "points100": true/false,
    "additionalPoints": 0-10,
    "grants": null/"50%"/"100%",
    "tuitionDiscount": null/"10%"/"50%",
    "priority": true/false,
    "earlyAdmission": true/false
  },


  "relevantDirections": ["экономика", "бизнес", "менеджмент", "финансы"],


  "dates": {

    "event": "2026-11-15",

  },

═══════════════════════════════════════════════════════════════════════
📌 ВАЖНЫЕ ПРАВИЛА
═══════════════════════════════════════════════════════════════════════


1. targetAudience:
   ✅ ПРАВИЛЬНО: ["8 класс", "9 класс", "10 класс", "11 класс"]
   ❌ НЕПРАВИЛЬНО: ["школьники", "абитуриенты", "10-11 класс"]


2. benefits - ВСЕГДА все 7 полей:
   {
     "bvi": false,
     "points100": false,
     "additionalPoints": 0,
     "grants": null,
     "tuitionDiscount": null,
     "priority": false,
     "earlyAdmission": false
   }


3. format - ТОЛЬКО следующие значения:
   - "очно"
   - "онлайн"
   - "выездное"
   - "смешанное_очно_онлайн"
   - "очно_выездное"


4. relevantDirections - возможные значения:
   - "экономика"
   - "бизнес"
   - "менеджмент"
   - "финансы"
   - "IT"
   - "все направления"


5. dates - форматы:
   - Одна дата: "2026-09-15"
   - Диапазон: "2026-09-01 - 2026-10-31"
   - С временем: "2026-09-15 18:00"


6. cost.type - ТОЛЬКО:
   - "бесплатно"
   - "платно"
   - "частично_платно"

7. Ссылки на ресурсы должны вести не на общий сайт, а на страницу самого события.
   Например,  событие "один день в РЭШ"   его ссылка - [https://openday.nes.ru/one-day-nes](https://openday.nes.ru/one-day-nes)   и её нужно использовать. НЕ использовать [https://www.nes.ru/](https://www.nes.ru/)

8. Если ссылка ведёт на страницу, на которой перечислено несколько событий например несколько подкастов,ТО карточка должна быть не про конкретный подкаст, а обобщающая про подкасты.

9. Описание карточки ОБЯЗАНО соответствовать тому, что увидит пользователь переходя по ссылке


═══════════════════════════════════════════════════════════════════════
🔍 ЧТО ИСКАТЬ ДЛЯ КАЖДОГО РЕСУРСА
═══════════════════════════════════════════════════════════════════════


Для каждого ресурса ОБЯЗАТЕЛЬНО определить:


1. ЛЬГОТЫ (benefits):
   ☑ БВИ - дает ли прием без вступительных испытаний?
   ☑ points100 - приравнивается ли к 100 баллам ЕГЭ?
   ☑ additionalPoints - сколько доп. баллов (0, 3, 5, 8, 10)?
   ☑ grants - есть ли гранты и какой % (50%, 100%)?
   ☑ tuitionDiscount - скидки на обучение?
   ☑ priority - дает ли приоритет при зачислении?
   ☑ earlyAdmission - досрочное зачисление?


2. СВЯЗЬ С НАПРАВЛЕНИЯМИ:
   - Как ИМЕННО этот ресурс помогает поступить на бизнес/экономику/менеджмент/финансы?
   - Указать в relevantDirections конкретные направления


3. ДОСТУПНОСТЬ:
   - Кто может участвовать? (классы)
   - Есть ли отбор? (selectionProcess)
   - Сколько мест? (participantLimit)


4. СРОКИ:
   - Регистрация
   - Проведение
   - Дедлайны
   - ВАЖНО!  если мероприятие до 26.01.2026  то его не учитываем и пропускаем, так как оно уже состоялось.


5. СТОИМОСТЬ:
   - Бесплатно/платно
   - Точная сумма
   - Возможность грантов
