# команда запуска и обновления ресурсов

**запускается только из Claude Code**

---

Обработай университет [НАЗВАНИЕ] полностью:

1. Создай резервную копию data/universities/hse/ в backups/hse_20260123/

2. Обработай каждую категорию последовательно:
   - infoEvents.json
   - olympiads.json
   - courses.json
   - educationalEvents.json
   - practicalEvents.json
   - summerPrograms.json
   - schools.json
   - onlineResources.json

3. Для каждого файла:
   - Прочитай все ресурсы
   - Провалидируй каждый (загрузи сайт, проверь описание)
   - Если статус NEEDS_UPDATE или MISMATCH:
     - Обнови description на suggested_description
   - Сохрани обновленный JSON файл

4. Создай отчет validation_summary_hse.json с результатами
